## Braiins Pool Setup

1. **Create Account:** Register a [fresh account](https://pool.braiins.com/). *(Do not use an existing one!)*
2. **Add Profile:** Go to [Account > Access Profiles](https://pool.braiins.com/settings/access) and create a new profile.
3. **Generate Token:**
    - Set permissions to *Read-only*.
    - Check *Allow access to web APIs*.
    - Click *Generate new token*.
4. **Paste:** Copy the token and paste it as your **Access Token**.
5. **Save:** Finally, click *Create Access Profile*.
