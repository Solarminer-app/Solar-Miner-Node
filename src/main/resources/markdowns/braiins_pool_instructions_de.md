## Braiins Pool Einrichtung

1. **Account erstellen:** Registriere einen [neuen Account](https://pool.braiins.com/). *(Bitte keinen bestehenden nutzen!)*
2. **Profil anlegen:** Navigiere zu [Account > Access Profiles](https://pool.braiins.com/settings/access) und erstelle ein neues Profil.
3. **Token generieren:**
    - Rechte auf *Read-only* setzen.
    - Haken bei *Allow access to web APIs* setzen.
    - Auf *Generate new token* klicken.
4. **Einfügen:** Token kopieren und hier als **Access Token** eintragen.
5. **Speichern:** Abschließend auf *Create Access Profile* klicken.
