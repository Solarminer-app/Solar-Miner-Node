## Verbinden der PV Anlage über Modbus TCP ##

### Hinweis Modbus TCP ###
Viele heutige Wechselrichter bieten über eine Schnittstelle namens Modbus TCP an.

Bitte stelle sicher, dass der Hersteller mehrere gleichzeitige Schnittstellen auf Modbus TCP zulässt.

Wenn nicht, verbinde deine Anlage bitte über eine andere Möglichkeit!

### Hinweis Config ###
Jedes Modbus Register ist anders.
Zu diesem Zweck können Nutzer Modbus Konfigurationen erstellen und bereitstellen.
Sollte keine Konfiguration für dein Gerät existieren, kannst du auf das
Zahnrad klicken und eine eigene erstellen oder eine bereits existierende importieren.
